Notes

  1. Video should be delivered as mp4 at a size of 280 (w) x 300 (h).
  2. Performace is critical. Each video should be no longer than 10-15 seconds in length (the shorter the better).
  3. Video should strive to meet a file size no greater than 1.5MB each (and that is probably pushing it). Remember, each video will add to the cumulitive weight of the page, so size must be kept low.
  4. Video utilizes prefers-reduced-motion in addition to the pause button. When user disables animation in their operating system, these videos will not autoplay.
  5. Cookie set or removed when pause button toggled to remember to desired setting.
  6. Video may not play if mobile device is in low-power mode. This is for a good reason—so video does not drain the battery life.
  7. Content or media grid items can be made interactive if desired.
