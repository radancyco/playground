Notes

  1. Video should be delivered as webm (or mp4) at a size of 280 (w) x 300 (h).
  2. Performace is critical. Each video should be no longer than 10-15 seconds in length (the shorter the better).
  3. Video should strive to meet a file size no greater than 1.5MB each (and that is probably pushing it). Remember, each video will add to the cumulitive weight of the page, so size must be kept low. Would not reccommend more than two rows of video for this reason.
  4. Each video will require a thumbnail sized to 302 x 312 and be deleivered as webp.
  5. Video utilizes prefers-reduced-motion in addition to the pause button. When user disables animation in their operating system, these videos will not autoplay.
  6. Cookie set or removed when pause button toggled to remember the desired setting on return visits.
  7. Video may not play if mobile device is in low-power mode. This is for a good reason—so video does not drain the battery life.
  8. Grid blocks with text can be linked, but only link the heading. For example: <h3 class="enhanced-grid__subhdr"><a href="#">Scholarship</a></h3>
  9. Videos may never be linked.
