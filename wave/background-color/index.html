--- title: "WAVE Contrast Hack" ---

Here we have a div with a background color of #6495ED and a link color of #FFFFFF.

Hello World

Clearly this fails contrast and only meets a 2.97:1 ratio.

Contrast issues can be circumvented in WAVE by applying a opaque background color to the link, such as background-color: #00000001;. This is being applied to the code below. Note how WAVE flags the fist code block, but not the second.

Hello World
